Habilidades

HTML5

HTML5

Sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web.


CSS3

CSS3

Sigla CSS Cascading Style Sheets (Folhas de estilo em cascata), onde se define a forma dos elementos dando estilos para seu projeto web.


JavaScript

JavaScript

É uma linguagem de programação que permite a você implementar itens complexos em páginas web. Juntamente com HTML e CSS, é uma das três principais tecnologias da web.


JQuery

JQuery

O objetivo do jQuery é facilitar muito o uso do JavaScript em seu site. O JQuery pega muitas tarefas comuns que requerem muitas linhas de código JavaScript para serem realizadas e as agrupa em métodos que você pode chamar com uma única linha de código.


Node js

Node js

Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web. A principal característica do Node.js é sua arquitetura assíncrona e orientada por eventos.


Bootstrap

Bootstrap

É uma estrutura de desenvolvimento front-end gratuita e de código aberto para a criação de sites e aplicativos da web. Projetado para permitir o desenvolvimento responsivo de sites mobile-first, o Bootstrap fornece uma coleção de sintaxe para designs de modelo.


React js

React js

O É a biblioteca mais popular do JavaScript e é usada para construir uma interface de usuário (IU). Ela oferece uma resposta excelente para o usuário adicionar comandos usando um novo método de renderizar sites.


Materialize

Materlialize

Material Design é um sistema unificado que combina teoria, recursos e ferramentas para criar experiências digitais, desenvolvido pela equipe de designers do Google com intuito de unificar a experiência do usuário na usabilidade dos seus softwares, seja no smartphone, tablets ou desktops.


UI / UX

UI / UX

O UI trata dos elementos com os quais o usuário interage, ao passo que o UX procura entender os comportamentos e emoções dessas pessoas no uso do produto. Entrevistas com usuários para entender necessidades, construção de personas e elaboração da jornada do usuário fazem parte do dia a dia de quem trabalha com UX.


Sass

Sass

É uma linguagem de extensão do CSS, a sigla significa “Syntactically Awesome Style Sheets” traduzindo ao pé da letra, folhas de estilo com uma sintaxe incrível. A sua ideia é adicionar recursos especiais como variáveis, mixins, funções e operações e outras opções variadas.


Git

Git

É um sistema de controle de versões distribuído, usado por um número impressionante de projetos e desenvolvimento de software, depende do Git para controle das versões, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.


GitHub

GitHub

É uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.
